# MC SWEP

This repository contains the native mesh DLL and Minecraft Java Edition world import tools for MC SWEP.

## Native DLL

If you are not using Garry's Mod's 64-bit version, download `gmcl_mcswep_native_mesh_win32.dll`.

If you are using the 64-bit version, download `gmcl_mcswep_native_mesh_win64.dll`.

Place the DLL in:

```text
GarrysMod/garrysmod/lua/bin/
```

DLL downloads are available on the [Releases page](https://github.com/schweiziske/mc_swep/releases/tag/standard).

## Import Tools

Minecraft Java Edition world import tools are available in the [`tools` folder](tools/).

See [`tools/README.md`](tools/README.md) for usage instructions.
